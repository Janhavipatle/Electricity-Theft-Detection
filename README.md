# Electricity Theft Detection

AI & IoT based electricity theft detection system using ESP32, Flask & MQTT.  

## Files
- `app.py` : Flask backend server
- `dashboard.html` : Frontend dashboard
- `esp32code.ino` : Code for ESP32 microcontroller

## How to Run
1. Run `app.py` using Python 3.
2. Open `dashboard.html` in browser.
3. Upload `esp32code.ino` to ESP32 board.
